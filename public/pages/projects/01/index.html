Implementation

A full-stack Discord application bringing the "One Lie Per Line" word game natively to Discord voice channels. This project is structured as a pnpm v10 monorepo and includes the interactive Activity frontend, an Express backend, a Discord.js utility bot, and a persistent SQLite database.

Github: https://github.com/SurrealArts/discord-lirdle-activity

Live demo: https://discord.com/oauth2/authorize?client_id=1495430267274264628

Tech Stack

  • Workspace: pnpm v10, monorepo
  • Frontend/Activity: HTML, CSS, Vanilla JS, `@discord/embedded-app-sdk`
  • Backend: Node.js 22, Express
  • Bot: `discord.js`, `canvas` (libcairo2/libpango native builds)
  • Database: SQLite, Prisma (`@prisma/adapter-better-sqlite3`)
  • Deployment: Docker, Docker Compose, Cloudflared
